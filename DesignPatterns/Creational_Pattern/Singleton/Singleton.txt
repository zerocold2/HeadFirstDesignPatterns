# Singleton:
	Is a class only instantiate once
# Singleton vs Global variable or static object
	When we assign an object to global variable, then you have to create that object at the begining of the application. in singleton we can instatiate the object when we want to use
# When to Use:
  - Object used for thread pools
  - caches
  - dialog boxes
  - registry setting
  - logging
  - and objects that act as device drivers to devices like printers and graphic cards
